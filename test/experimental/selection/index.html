Selected Text test

Remove selection

Monocle needs a set of events for "some text has been selected", "the selection has changed", and "selection deselected". This is so we can build contextual menus for these cases.

Furthermore, as soon as a page begins turning, any selected text should be deselected. This is trivial in most browsers — window.getSelection().removeAllRanges() — but bizarrely difficult in MobileSafari.

The Remove Selection button is wired to take effect on a tap or a swipe, on touch-based and cursor-based browsers.
